Bem-vindo ao App-site!

Este é um projeto Angular inicial, criado como um boilerplate para futuras aplicações. O objetivo é fornecer uma estrutura de pastas organizada e escalável, seguindo as melhores práticas de desenvolvimento Angular.

Estrutura do Projeto

  • /core: Contém serviços singleton, interceptors e guards.
  • /features: Onde cada funcionalidade ou "página" da aplicação reside.
  • /shared: Módulos, componentes, diretivas e pipes reutilizáveis.

Integração com a API

Este frontend está integrado a uma API de backend .NET, que gerencia os dados de usuários. A comunicação é feita através de serviços Angular que consomem os endpoints RESTful da API.
